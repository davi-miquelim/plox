# LOX interpreter implemented in Python
This is a simple implementation of the LOX interpreter in Python.

## Running the code
To run the code, simply run the following command in the terminal:
```bash
python3 lox.py
```

## Example
```bash
python3 lox.py
> var a = 10;
> print a;
10
> var b = 20;
> print a + b;
30
> var c = "Hello, World!";
> print c;
Hello, World!
```
